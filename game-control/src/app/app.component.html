Game-Control component

  1. The GameControl Component has buttons to start and stop the game
  2. When starting the game, an event should get emitted each second
  3. When stopping the game, no more events should get emitted
  4. A new Odd component should get created for every odd number emitted, should output Odd - NUMBER in red
  5. A new Even component should get created for every even number emitted, should output Even - NUMBER in red
